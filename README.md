#Project

This is Game 357. Try to beat the computer.

CLI:

    ./bin/debug/cpp-project

Unit tests:

    ./bin/debug/cpp-project-tests

##System requirements

    Linux
    Mingw
    C++11 g++ or clang

##Compile

Build:

    make

Release:

    make RELEASE=on

Verbose:

    make VERBOSE=on

Clean:

    make clean
